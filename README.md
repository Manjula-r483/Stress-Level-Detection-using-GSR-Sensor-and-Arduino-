# Stress Level Detection using GSR Sensor and Arduino

## 📌 Project Overview
This project detects human stress levels using a GSR (Galvanic Skin Response) sensor.
It measures changes in skin resistance caused by sweat gland activity during stress.

## 🎯 Objective
To design a low-cost stress monitoring system using Arduino for basic emotional analysis.

## 🛠 Components Used
- Arduino Uno
- GSR Sensor
- Resistors
- Breadboard
- Jumper wires
- LCD Display (optional)

## ⚙ Working Principle
When a person experiences stress, sweat secretion increases.
This reduces skin resistance.
The GSR sensor captures this variation.
Arduino processes the signal and determines stress level.

- Embedded C Programming
